# Task-Manager
---
## Functionality

This is a task manager that allows you to create, visualize, edit and complete or delete tasks.
Tasks are saved on a JSON file to keep the information even after closing the program.
---
## Characteristics 
- Visualize tasks
- Add new tasks
- Mark tasks as completed (by name or by ID)
- Edit tasks (name or status)
- Delete tasks (by name or ID)
- Automatic saving to tasks.json file

--- 

## Requirements
- Python 3.x
